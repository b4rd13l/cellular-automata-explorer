# cellular-automata-explorer
Explorador interactivo de reglas de autómatas celulares 1D (0-255) con visualización y animación.

Características:
- Visualización de la evolución de autómatas celulares
- Animación de generaciones
- Tabla de explicación de reglas
- Interfaz gráfica de usuario interactiva con Tkinter
